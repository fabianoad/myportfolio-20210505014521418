Instalacao ReactJs

$ create-react-app websitefabiano

e para testar a instalacao:

$ npm start

Ajustar App.js

Escolher tema Bootstrap para incorporar ao React

Tema escolhido: startbootstrap-resume-gh-pages.zip

Add folder to workspace

Realizar copias das estruturas de pastas

De bootstrap-theme para a estrutura React

assets, css e js para public

copiar links e scripts do index.html do bootstrap-theme para o index do public React

Colocar %PUBLIC_URL% no caminho dos links e scripts copiados

Ex: script src="%PUBLIC_URL%/js/scripts.js"

Instalar o React Router

$ npm install --save react-router-dom

Criar o componente PageWrapper.js

Criar pasta "components" conforme estrutura proposta nesse documento

Criar arquivo PageWrapper.js nessa pasta

Conteudo desse arquivo sera o nav(Navigation) do index.html do bootstrap-theme

Importar o Link de react-router-dom

Trocar "href" por "to" e "a" por "Link"

E no final do da classe colocar {this.props.children}, de onde virao os outros componentes com a nav em comum
